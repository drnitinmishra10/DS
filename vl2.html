Lab Experiment – 2

Introduction to Kaggle Platform and Basic Data Handling

Prepared By: Dr. Nitin Mishra
Professor, Department of Computer Science & Engineering

Experiment Aim

To familiarize students with the Kaggle platform, upload datasets, perform basic Pandas operations, and read different types of datasets using Python in Kaggle Notebook.

Learning Outcomes

After completing this experiment, students will be able to:
  • Create and access a Kaggle account.
  • Create a Kaggle Notebook.
  • Add datasets to a Kaggle notebook.
  • Import the Pandas library.
  • Load and explore datasets.
  • Read CSV, Excel, JSON and Text files using Pandas.
  • Perform basic data inspection operations.

Software Requirements

  • Google Account
  • Kaggle Account
  • Internet Connection
  • Web Browser

Theory

Kaggle is one of the world's largest platforms for Data Science, Machine Learning, and Artificial Intelligence. It provides free cloud-based Jupyter notebooks, GPU support, datasets, competitions, and collaborative projects. Students can perform data analysis without installing Python on their local computers.

Lab Tasks

Task 1: Login to Kaggle using Google ID

  1. Open https://www.kaggle.com
  2. Click Sign In.
  3. Select Continue with Google.
  4. Complete profile setup.
  5. Create a new Notebook.

Task 2: Add Dataset to Kaggle Notebook

  1. Create a new notebook.
  2. Click Add Data.
  3. Search any public dataset (Titanic, Iris, House Prices etc.).
  4. Add the dataset.
  5. Verify that the dataset appears under the Input folder.

Task 3: Basic Pandas Code in Kaggle

import pandas as pd

df = pd.read_csv('/kaggle/input/your-dataset/data.csv')

print(df.head())

print(df.tail())

print(df.shape)

print(df.columns)

print(df.info())

print(df.describe())

Task 4: Reading Different Types of Files

1. Reading CSV File

import pandas as pd

df = pd.read_csv("sample.csv")
print(df.head())

2. Reading Excel File

df = pd.read_excel("sample.xlsx")
print(df.head())

3. Reading JSON File

df = pd.read_json("sample.json")
print(df.head())

4. Reading Text File

df = pd.read_csv("sample.txt", delimiter="\t")
print(df.head())

Basic Pandas Commands

Command Description
head() Display first five rows
tail() Display last five rows
shape Returns rows and columns
columns Shows column names
info() Dataset information
describe() Statistical summary
isnull().sum() Count missing values
value_counts() Frequency of values
