Lab Experiment – 1

Study of Different Tools Used in Data Science

Prepared By:
Dr. Nitin Mishra
Professor, Department of Computer Science & Engineering

Experiment Aim

To study the commonly used tools in Data Science and understand their purpose, features, and practical applications.

Learning Outcomes

After completing this experiment, students will be able to:
  • Identify major Data Science tools.
  • Understand the purpose of each tool.
  • Compare different Data Science platforms.
  • Select appropriate tools for various Data Science tasks.
  • Explore cloud-based and desktop development environments.

Theory

Data Science involves collecting, processing, analyzing, visualizing, and modeling data. Various software tools are available for performing these tasks efficiently. These tools support programming, data visualization, machine learning, deep learning, big data analytics, and cloud computing.

Software Requirements

  • Internet Connection
  • Google Chrome / Edge Browser
  • Python (Optional)
  • Anaconda (Optional)

Observation

  • Different Data Science tools provide different functionalities.
  • Cloud platforms eliminate the need for software installation.
  • Jupyter Notebook is widely used for experimentation.
  • Kaggle provides datasets and competitions.
  • Google Colab provides free GPU support.

Result

The experiment was successfully completed. Students explored the major tools used in Data Science and understood their applications in programming, data analysis, visualization, and machine learning.
