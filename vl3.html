Aim

To perform basic data manipulation operations using the Pandas library, including loading datasets, selecting data, filtering records, modifying datasets, and handling missing values.

Learning Outcomes

After completing this experiment, students will be able to:
  • Load datasets using Pandas.
  • Explore DataFrames.
  • Select rows and columns.
  • Filter data using conditions.
  • Add, modify and delete columns.
  • Handle missing values.
  • Sort datasets.

Software Requirements

  • Python 3.x
  • Pandas Library
  • Kaggle Notebook / Google Colab / Jupyter Notebook
  • CSV Dataset (Titanic/Iris/Employee Dataset)

Theory

Data manipulation is the process of organizing, transforming, filtering, and modifying data before analysis. Pandas is one of the most powerful Python libraries that provides flexible data structures such as Series and DataFrame to efficiently perform these operations. Data manipulation improves data quality and prepares datasets for machine learning and visualization.

Lab Tasks

Task 1: Load and Explore Dataset

Import the Pandas library and load a CSV dataset. Display the first few records and basic information about the dataset.

import pandas as pd

df = pd.read_csv("employee.csv")

print(df.head())
print(df.tail())
print(df.shape)
print(df.columns)
print(df.info())
print(df.describe())

Task 2: Select and Filter Data

Select specific rows and columns and filter records based on conditions.

# Selecting Columns
print(df["Name"])
print(df[["Name","Salary"]])

# Selecting Rows
print(df.loc[0:4])

# Filtering Data
print(df[df["Salary"] > 50000])

print(df[(df["Age"] > 30) &
         (df["Department"] == "IT")])

Task 3: Modify Dataset

Add new columns, modify existing values, rename columns, and delete unnecessary columns.

# Add Column
df["Bonus"] = df["Salary"] * 0.10

# Modify Column
df["Salary"] = df["Salary"] + 5000

# Rename Column
df.rename(columns={"Salary":"MonthlySalary"},
          inplace=True)

# Delete Column
df.drop("Bonus", axis=1, inplace=True)

print(df.head())

Task 4: Handle Missing Values and Sort Data

Identify missing values, replace or remove them, remove duplicate records, and sort the dataset.

# Missing Values
print(df.isnull().sum())

df.fillna(0, inplace=True)

# Remove Duplicates
df.drop_duplicates(inplace=True)

# Sort Data
df.sort_values(by="Age", inplace=True)

print(df.head())

Common Pandas Functions Used

Function Description
read_csv() Loads CSV dataset
head() Displays first five rows
tail() Displays last five rows
shape Returns number of rows and columns
info() Displays dataset information
describe() Shows statistical summary
loc[] Select rows and columns using labels
drop() Deletes rows or columns
rename() Renames columns
fillna() Replaces missing values
drop_duplicates() Removes duplicate records
sort_values() Sorts the dataset
